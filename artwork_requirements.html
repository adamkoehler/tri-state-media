Artwork Requirements

Only 1 label per file please.
Please check all artwork for typing and spelling mistakes – we do NOT proofread artwork. We will gladly provide a pdf proof for your viewing.

Acceptable file formats are AI (Adobe Illustrator), PSD (Adobe Photoshop), EPS (Encapsulated Postscript) and PDF (Adobe’s Portable Document Format) – if you need help choosing the correct format, call us and we’ll be glad to help.

ALL fonts should be outlined (or rasterized). Type should be no smaller than 4 pt. in size and not condensed more than 10%. (6 pt. for reverse copy size).

Include all images (photos should be 100% size or 1:1 and 300 dpi when placed, to keep files manageable) with appropriate bleeds (standard is .125”).

All files should be designed and submitted in CMYK mode (not RGB) – otherwise significant color changes can occur when file is converted to CMYK. Ink colors are called out (c=, m=, y=, k=,) for process colors and PMS (Pantone) numbers for spot colors.

Please remove any unused colors and layers from file. (If white is to print please indicate so on art file). Dielines, fold lines, dimensions, crop marks and registration marks should be on separate layers marked “do not print”.

Radius corner dimensions should be indicated.

Barcode Specifications:

Barcodes should be to size with .125″ quiet zone surrounding the barcode (Industry Standard). Barcode will scan best if printed in black.

Please provide the correct number for the barcode. If Tri-State Media is to create barcode, please provide the barcode symbology and characters.

Contact Tri-State Media for information on where to send file(s) or with any questions. 513-933-0101
